# Visa Document Processing System

## Project Overview
A comprehensive visa document processing system helping Bangladeshi travelers with visa applications. Version 1 focuses on Iceland tourist visa processing.

## Version 1 Scope
- **Country:** Iceland (Schengen Visa)
- **Visa Type:** Tourist Visa
- **Core Features:**
  - Document upload interface
  - AI-powered document analysis
  - Missing document generation
  - Complete visa package download

## Tech Stack
- **Backend:** Python 3.10+ with FastAPI
- **Frontend:** React 18+ with Material-UI
- **Database:** PostgreSQL 14+
- **AI:** Google Gemini API
- **PDF Processing:** PyPDF2, reportlab, pytesseract

## Required Documents for Iceland Tourist Visa

### Documents Users Provide:
1. Passport copy
2. NID (Bangla version)
3. Visa history copy
4. TIN certificate
5. Income tax returns (last 3 years)
6. Asset valuation certificate
7. Hotel booking confirmation
8. Air ticket booking
9. Bank solvency statements (multiple types)

### Documents System Generates:
1. NID English translation
2. Professional visiting card
3. Cover letter
4. Travel itinerary
5. Travel history summary
6. Home tie statement letter
7. Financial statement summary
